What do the following commands do? 
a) date 
Shows current date and time
b) touch 
Creates a new file in the file system
c) wc  
Used to count the number of lines, words, and characters in a file
d) What is the meaning of each output of the command ‘wc’? 
Hint: Use manpage if you are unsure. 
Lines: This shows the number of lines in the file.

Words: This shows the number of words in the file.

Characters: This shows the number of characters in the file.
